## HiraPedia

## サイト概要
枚方を愛する人たちのためのサイト

### サイトテーマ
このサイトでは、枚方や枚方近郊にある
自分にとっての思い出の場所、お気に入りの場所や、
みんなに是非ともお勧めしたい場所（お店、景色、公園など）や、
普通知らないような穴場スポットのような所などを投稿することが出来て、
枚方好きな人たちと共有しあえるというサイトです。

### テーマを選んだ理由
枚方のことをもっと知りたい、また知って欲しいからです。

### ターゲットユーザ
枚方や枚方近郊で、
生まれ育った人、
働いていて馴染みがある人、
そんな人たちを対象としています。

### 主な利用シーン
枚方のことをもっと知りたい時、枚方のことをもっと知って欲しい時
